Prioritize Network Metrics ()
The activities associated with the determination which performance gaps will be addressed in solution design. The purpose of this prioritization process is to ensure the organization is focused on the metrics that have (the highest) impact on the overall performance gap (i.e. cascading the performance goals into the organization).
