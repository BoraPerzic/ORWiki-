Process Return Inquiry ()
The activities associated with receiving, verifying and responding to an inquiry about the terms and conditions for product returns. This may include collecting case specific information, initial problem identification and offering options to the customer. An Inquiry may result in (trigger) theS402process. Process Return Inquiryis aReverse Logisticsprocess.
